Privacy Policy

Last Updated: February 23, 2026

Your Privacy Matters: This Privacy Policy explains how Inasha collects, uses, stores, and protects your information when you use our Discord bot. We are committed to maintaining minimal data collection and protecting your privacy.

1. Introduction

Inasha ("we", "our", "the Bot") is a Discord bot that provides moderation, utility, and entertainment features for Discord servers. This Privacy Policy describes how we handle data when you use Inasha.

By using Inasha, you consent to the data practices described in this policy.

2. Information We Collect

Inasha collects minimal data necessary to provide its services. We collect information in the following categories:

2.1 Automatically Collected Data

When you use Inasha, we automatically collect:

Data Type Purpose Retention
Server IDs Identify which servers use the Bot, enable server-specific features While Bot is in server
Channel IDs Server statistics tracking, channel management features While feature is enabled
User IDs Command execution, game features, moderation logging Temporary (command duration)
Server Member Counts Display server statistics Real-time updates only
Channel Counts Display server statistics Real-time updates only

2.2 Command Usage Data

When you interact with Inasha through commands:

  • Command Names: Which commands are being used (for analytics and improvement)
  • Command Parameters: Only processed temporarily to execute the command
  • Interaction Timestamps: When commands were executed (for rate limiting and abuse prevention)

2.3 Game and Fun Feature Data

For certain features, we temporarily store:

  • Fish Leaderboard Data: User IDs and fish counts for the fishing game
  • Game Results: Outcomes of commands like ship, family, gay, freaky (processed temporarily)

Note: This data is stored locally in memory or in simple JSON files and is never shared with third parties.

2.4 Server Statistics Configuration

If you enable server statistics tracking:

  • Server ID and configured channel IDs for member and channel count displays
  • This data is stored in serverstats.json to maintain your configuration

3. Information We Do NOT Collect

Inasha does not collect:

  • Message Content: We do not log or store the content of your messages (except when you directly invoke a command)
  • Voice Data: We do not access or store voice communications
  • Personal Information: We do not collect names, email addresses, phone numbers, or other personal identifiers beyond Discord User IDs
  • Payment Information: Inasha is free and does not process any payment data
  • Location Data: We do not collect or track your location
  • Device Information: We do not collect information about your device

4. How We Use Your Information

We use the collected data solely to:

  • Provide Bot Functionality: Execute commands, display server information, run games
  • Maintain Server Statistics: Update member and channel count displays
  • Improve the Bot: Understand which features are used to guide development
  • Prevent Abuse: Implement rate limiting and detect malicious use
  • Respond to Support Requests: Help troubleshoot issues when you contact us

We do NOT use your data for:

  • Marketing or advertising
  • Selling or sharing with third parties
  • Profiling or tracking across other services

5. Data Storage and Security

5.1 Where Data is Stored

  • Server Configuration: Stored locally on our server in JSON files
  • Temporary Data: Stored in memory during command execution and cleared afterward
  • No Cloud Storage: We do not use external cloud databases for user data

5.2 Security Measures

We implement reasonable security measures to protect data:

  • Access to the Bot's server is restricted to authorised maintainers
  • Data files are stored with appropriate file system permissions
  • We use Discord's official API and follow their security best practices
  • No data is publicly accessible or shared with unauthorised parties

5.3 Data Retention

  • Active Servers: Configuration data is retained while Inasha is in your server
  • Removed Servers: When Inasha is removed from a server, associated data is deleted within 30 days
  • Command Data: Processed in real-time and not permanently stored
  • Logs: Basic error and system logs may be kept for up to 30 days for debugging purposes

6. Data Sharing and Third Parties

We do NOT sell, trade, or share your data with third parties, except:

  • Discord: All data is processed through Discord's API as required for the Bot to function
  • Legal Requirements: We may disclose data if required by law, court order, or government request
  • Service Protection: We may share data to prevent fraud, abuse, or security threats

Inasha does not use analytics services, advertising networks, or other third-party data processors.

7. Discord's Role

Inasha operates on Discord's platform and is subject to Discord's Privacy Policy. Discord may collect additional data about your use of their service and Inasha. Please review Discord's Privacy Policy for more information.

8. Your Rights and Choices

8.1 Access and Deletion

You have the right to:

  • Request Access: Ask what data we have collected about you
  • Request Deletion: Ask us to delete your data
  • Opt-Out: Stop using Inasha or remove it from your server at any time

To exercise these rights, contact us through GitHub or our support server.

8.2 Server Owner Controls

If you own a Discord server with Inasha:

  • You can remove the Bot at any time to stop data collection
  • You can disable specific features (like server statistics) using commands
  • You control which roles and users can access Bot commands through Discord permissions

9. Children's Privacy

Inasha is not intended for children under 13 years of age (or the minimum age required in your country to use Discord). We do not knowingly collect data from children. If you believe a child has provided data to Inasha, please contact us immediately.

10. International Users

Inasha is hosted on servers that may be located in different countries. By using the Bot, you consent to the transfer and processing of your data in these locations.

11. Automated Responses

Inasha includes automatic response features (e.g., responding to "meow" or dog-related words). These responses are triggered by keyword detection in messages visible to the Bot. No message content is logged or stored - responses are generated in real-time.

12. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. When we make changes:

  • The "Last Updated" date will be revised
  • Significant changes will be announced in our support server
  • Continued use of Inasha constitutes acceptance of the updated policy

We encourage you to review this Privacy Policy periodically.

13. Contact Us

If you have questions about this Privacy Policy or want to exercise your data rights, please contact us:

  • GitHub: github.com/MrLuckyGamer/inasha-discord-bot
  • Support Server: Join our Discord support server for direct assistance

14. Data Breach Notification

In the unlikely event of a data breach that affects your information, we will:

  • Investigate the breach promptly
  • Notify affected users through our support server or GitHub
  • Take steps to prevent future breaches
  • Comply with applicable data breach notification laws

Summary: Inasha collects minimal data (server IDs, user IDs for commands, and optional server statistics) solely to provide its features. We do not sell your data, track you across services, or collect personal information beyond what's necessary for the Bot to function. You can stop using Inasha at any time by removing it from your server.
